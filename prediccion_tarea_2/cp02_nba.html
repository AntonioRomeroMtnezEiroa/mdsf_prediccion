Diccionario de variables

Variable endgena: - Salario: log

Variables exgenas:
- Edad (Age): se presupone que a mayor edad mayor salario - Edad elevado alcuadrado: considero que a partir de cierta edad ya no aumenta el salario con la edad - Nmero del draft(NBA_DraftNumber): a menor nmero en el draft mayor salario - Minutos jugados (MP): a mayor nmerode minutos jugados mayor salario - Minutos jugados al cuadrado: a partir de un cierto nmero de minutosjugados ya no aumenta el salario - Eficiencia del jugador: a mayor eficiencia mayor salario - Eficiencia deljugador al cuadrado: a partir de cierto nivel de eficiencia ya no afecta al salario - Contribucin a las victorias del equipo: a mayor contribucin a las victorias del equipo mayor salario - Contribucin a las victorias del equipo al cuadrado: a partir de cierto nivel de aportacin a las victorias del equipo ya no afecta al salario - Porcentaje de participacin en el juego (USG%): A mayor participacin mayor salario - Valor sobre jugadorde reemplazo (VORP): a mayor VORP mayor salario - Valor sobre jugador de reemplazo al cuadrado: a partir de cierto nivel de VORP ya no afecta al salario - Efectividad de tiro (TS%): a mayor efectividad de tiro mayor salario - Efectividad asistencias (AST%): a mayor efectividad de asistencias mayor salario - Interaccinde WS y VORP (WS:VORP): considero que estn relacionadas estas dos variables, a mayores valores deWS y VORP mayor ser el salario del jugadorA continuacin se filtra la base de datos para poder observar slo las variables que me interesan.
